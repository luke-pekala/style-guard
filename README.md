# StyleGuard

Documentation style checker for teams that care about craft.

🔗 [styleguard.vercel.app](https://styleguard.vercel.app)

Paste your documentation and StyleGuard flags passive voice, weak verbs, filler words, and sentences that are too long. Click any violation to see a fix suggestion.

## Features
- 35 style rules across four categories
- Inline fix suggestions on click
- Score out of 100
- Dark and light mode
- Drag and drop .txt / .md / .rst files
- No login. No install. Runs in the browser.

## Stack
Vanilla HTML, CSS, JavaScript — hosted on Vercel.
