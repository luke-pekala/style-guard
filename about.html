01 About StyleGuard Pro

What it does

StyleGuard Pro analyses your documentation against a set of proven technical writing rules and returns colour-coded violations with specific fix suggestions. All analysis runs in your browser — no data leaves your machine.

Built as App 01 of the ai-engineer.app portfolio — a series of documentation and productivity tools grounded in 666 validated pain points from 25 technical writing books.

Detection rules

Passive voice
Detects passive constructions (is updated, was generated, were encountered) and flags them for active rewrites.
Weak verb
Flags overused weak verbs (utilize, leverage, facilitate, implement) and applies direct plain-English replacements on fix.
Jargon
Identifies technical jargon and buzzwords likely to confuse readers. Scored per paragraph in the density panel.
Long sentence
Flags sentences over 25 words. Visualised in the sentence heat map with low/medium/high colour coding.
Filler word
Catches meaningless padding phrases (it should be noted that, in order to, a significant number of) and applies deletions or replacements on fix.

Roadmap

v1.0 uses regex-based detection. The planned v2.0 will add Claude API integration for semantic analysis, custom style guide uploading, batch file processing, and a VS Code extension.
